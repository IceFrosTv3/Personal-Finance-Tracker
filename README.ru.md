# Lumincash — Личный финансовый трекер

Одностраничное приложение для учёта личных доходов и расходов. Позволяет управлять категориями, просматривать транзакции и анализировать бюджет через графики.

> [English documentation](./README.md)

---

## Стек технологий

- **HTML5** — разметка шаблонов
- **SCSS** — стилизация (с разбивкой на партиалы)
- **TypeScript** — логика приложения, роутинг, компоненты
- **Vite** — сборщик
- **npm** — менеджер пакетов
- **Bootstrap 5** + **Bootstrap Icons** — UI-компоненты и иконки
- **Chart.js** — круговые графики доходов и расходов
- **Tempus Dominus** — датапикер

---

## Установка и запуск

```bash
# Клонировать репозиторий
git clone https://github.com/IceFrosTv2/Personal-Finance-Tracker.git
cd frontend

# Установить зависимости
npm install
```

### Переменные окружения

Создайте файл `.env` в корне проекта:

```env
VITE_HOST="http://localhost:3000"
```

`VITE_HOST` — базовый URL сервера бэкенда.

```bash
# Запустить в режиме разработки
npm run dev

# Собрать для продакшена
npm run build
```

## Функциональность

- Регистрация и авторизация с клиентской валидацией форм
- Дашборд с балансом и круговыми графиками доходов и расходов
- Фильтрация по периоду: сегодня / неделя / месяц / год / всё время / произвольный диапазон
- Управление категориями доходов и расходов (создание, редактирование, удаление с подтверждением)
- Список транзакций с теми же фильтрами по периоду
- Создание и редактирование операций: тип, категория, сумма, дата, комментарий
- Баланс обновляется автоматически после любого действия с операциями
- Операции с удалёнными категориями отображаются как «без категории» на графиках
- Клиентский роутинг без перезагрузки страницы
- Адаптивная вёрстка: сворачиваемый сайдбар на десктопе, выдвижное меню на мобильных устройствах (< 650px)
- Тёмная тема (переключается кнопкой в сайдбаре, выбор сохраняется в localStorage)
- Постраничная навигация в списке транзакций (настраиваемый размер страницы, кнопки Назад / Вперёд)
